# 실습 프로젝트

## Nest JS 로 게시판 API 만들기

### ENV 예시
```
# Common environment variables
NODE_ENV=development
PORT=3000

# PostgresSQL
DB_TYPE=postgres
DB_HOST=localhost
DB_PORT=5432
DB_DATABASE=board
DB_USERNAME=root
DB_PASSWORD=1234
```


## 개발환경 실행
```
 yarn run start:dev
```

## 상용환경 실행
```
 yarn run start:prod
```
